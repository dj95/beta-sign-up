ETdb

- Spotting the home phoning martians in your app store.

We set the goal for our ETdb API to create a database which contains information about most of the popular apps and which APIs they call. We are going to start a beta test later in Q3 2017 when the main development is done. In order to participate in the closed beta program, please leave a subscription.


Participate

If you want to get more information what we run or if you want to take a part in the development feel free to check out the Github-Repository.

Software

API-Server

The API-Server is the heart of our project. It serves the database and the API which offers information about apps and features an interface to extend the database.
  • Language NodeJS
  • Database MongoDB

API-Client

The API-Client is your chance to work with our API. It offers the possibility to test the apps of your choice and publish the results in our database.
  • Language Python
  •  
  • Coming Soon

Frontend

The frontend is integrated into a website, so you are able to search through the database whenever you want, wherever you are.
  • Language HTML
  • Scripting JavaScript
