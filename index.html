ETdb

Spotting the home phoning martians in your app store.

We set the goal for our ETdb API to create a database which contains information about most of the popular apps and which APIs they call. We are going to start a beta test later in Q3 2017 when the main development is done. In order to participate in the closed beta program, please leave a subscription.

If you want to get more information what we run or if you want to take a part in the development feel free to check out the Github-Repository.
